bootstrap对talbe类做了非常小的定义，css代码如下

.table {
  border-collapse: collapse;
  border-spacing: 0;
}

这里的collapse意思是 如果可能，边框会合并为一个单一的边框。会忽略 border-spacing 和 empty-cells 属性。

额外的一些标识

active success info warning danger
Class 描述
.active 鼠标悬停在行或单元格上时所设置的颜色
.success 标识成功或积极的动作
.info 标识普通的提示信息或动作
.warning 标识警告或需要用户注意
.danger 标识危险或潜在的带来负面影响的动作
